It is an optimised way to get value of a number upto n times in logarithmic time.

The approach is simple simplify the power in terms of binary values initialise a value say tmp by the number and keep multiplying to itself for the given number of binary values and add the tmp to ans whenever 1 is fetched during iteration. 

public class binaryexponentiation
{
    public static void main(String[] args){
        int n=3,power=13;
        binaryexponentiation tt = new binaryexponentiation();
        System.out.println(tt.logarithmic(n,power));
        System.out.println(tt.multiplicative(n,power));
    }
    public long logarithmic(int n,int power){
        long ans=1;
        while(power>0){
            if(power%2==1){
                ans*=n;
            }
            power/=2;
            n*=n;
        }
        return ans;
    }
    public long multiplicative(int n,int power){
        if(power==0){
            return 1L;
        }
        long tmp=multiplicative(n,power/2);
        if(power%2==1){
            return n*tmp*tmp;
        }
        else{
            return tmp*tmp;
        }
    }
}
